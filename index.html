Sobre el proyecto

Infraestructura cívica digital para ciudades argentinas

Patentes Perdidas nació como respuesta concreta a un problema real: durante lluvias intensas en Santa Fe, cientos de patentes de vehículos se desprenden y no existe un sistema centralizado para recuperarlas.

Esta plataforma opera como infraestructura pública digital: gratuita, abierta y centrada en la comunidad. No reemplaza a los organismos oficiales — los complementa con velocidad y alcance comunitario.

⚠️

Aviso: Esta es una plataforma colaborativa. No verificamos la identidad de los usuarios. Siempre confirmá la información antes de contactar a alguien.

1. Crear la hoja de cálculo

Creá una hoja en Google Sheets con estas columnas exactas en la fila 1:

id | patente | estado | ciudad | zona | contacto | fecha | lat | lng | resuelto | notificar

2. Apps Script — API completa

// ================================================
//  PATENTES PERDIDAS v2 — Apps Script API REST
//  Soporta: GET, POST, PATCH (resolver patente)
// ================================================

const SHEET_NAME = "Patentes";

function doGet(e) {
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);
  const data = sheet.getDataRange().getValues();
  const headers = data[0];
  const rows = data.slice(1).filter(r => r[0]);

  const records = rows.map(row => {
    const obj = {};
    headers.forEach((h, i) => obj[h] = row[i] ?? "");
    return obj;
  });

  // Filtrar por ciudad si se pasa como query param
  const ciudad = e?.parameter?.ciudad;
  const filtered = ciudad
    ? records.filter(r => (r.ciudad || "").toLowerCase() === ciudad.toLowerCase())
    : records;

  return jsonResponse({ ok: true, data: filtered, total: filtered.length });
}

function doPost(e) {
  try {
    const body = JSON.parse(e.postData.contents);
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(SHEET_NAME);

    // PATCH: resolver patente
    if (body._method === "PATCH" && body.id) {
      return resolverPatente(sheet, body.id);
    }

    // POST: nueva publicación
    sheet.appendRow([
      body.id        ?? generateId(),
      body.patente   ?? "",
      body.estado    ?? "",
      body.ciudad    ?? "Santa Fe",
      body.zona      ?? "",
      body.contacto  ?? "",
      new Date().toLocaleString("es-AR"),
      body.lat       ?? "",
      body.lng       ?? "",
      "false",         // resuelto
      body.notificar ?? "false"
    ]);

    return jsonResponse({ ok: true, message: "Publicado correctamente" });

  } catch (err) {
    return jsonResponse({ ok: false, error: err.toString() }, 500);
  }
}

function resolverPatente(sheet, id) {
  const data = sheet.getDataRange().getValues();
  const headers = data[0];
  const idCol = headers.indexOf("id");
  const resolveCol = headers.indexOf("resuelto");

  for (let i = 1; i < data.length; i++) {
    if (data[i][idCol] == id) {
      sheet.getRange(i + 1, resolveCol + 1).setValue("true");
      return jsonResponse({ ok: true, message: "Marcado como resuelto" });
    }
  }
  return jsonResponse({ ok: false, error: "ID no encontrado" });
}

function jsonResponse(obj, code) {
  return ContentService
    .createTextOutput(JSON.stringify(obj))
    .setMimeType(ContentService.MimeType.JSON);
}

function generateId() {
  return Date.now().toString(36) + Math.random().toString(36).slice(2, 5);
}

3. Desplegar y conectar

  1. Apps Script → Implementar → Nueva implementación → Aplicación web
  2. Ejecutar como: Yo mismo | Acceso: Cualquier usuario
  3. Copiá la URL generada
  4. En app.js reemplazá: SHEETS_API_URL = "tu-url-aquí"
  5. Cambiá: USE_LOCAL_STORAGE = false

4. Publicar en GitHub Pages

  1. Subí los 3 archivos al repo (root)
  2. Settings → Pages → Source: main / (root)
  3. URL: tu-usuario.github.io/nombre-repo
